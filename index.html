Pinkboba is a CLI tool that scaffolds frontend UIs and Rust (Axum) backends. Generate 25 UI components, full CRUD APIs with JWT auth, SQLite, and more — all from the terminal.

⚡

Instant Scaffolding

Create a new project with a single command. Choose from 5 templates or generate a Rust backend with SQLite and JWT auth.

🧩

25 UI Components

Built-in components with pb- prefix. Add them individually via fb-component or install the glass package for 27 more.

🦀

Rust Backend (Axum)

Generate a complete Rust API with 11 bd-component commands: scaffold, model, controller, auth (bcrypt+JWT), middleware, response wrapper, pagination, and more.

🗄️

DB Admin Studio

Built-in SQLite database admin — browse tables, run queries, edit rows, and manage your schema from a web UI (pinkboba admin) or directly from the CLI.

📦

No Build Step Required

Frontend outputs plain HTML, CSS, and JS. No bundlers, no frameworks, no config. Just open index.html or deploy the Rust binary.

🎨

Dark Theme by Default

All components ship with a modern dark theme. Toggle between light and dark mode with the built-in theme component.

🌐

External Packages

Install component packages from GitHub with pinkboba add glass. The package system downloads components at runtime — no recompilation needed.

Quick Start — Frontend

Get a project running in under 30 seconds.

Create a project

pinkboba new my-app

Add components

pinkboba fb-component navbar && pinkboba fb-component hero

Preview

pinkboba serve → open http://localhost:8080

Quick Start — Full Stack

Add a Rust backend with SQLite, auth, and API routes.

Scaffold backend

pinkboba bd-component scaffold --with-db --with-env

Generate model + controller

pinkboba bd-component model User --fields name:String,email:String

pinkboba bd-component controller User --fields name:String,email:String

Add auth + serve

pinkboba bd-component auth && pinkboba serve

→ http://localhost:3000/api
