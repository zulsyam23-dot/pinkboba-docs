Pinkboba is a CLI tool that scaffolds professional web projects, generates glassmorphism UI components, and manages your frontend assets — all from the terminal.

⚡

Instant Scaffolding

Create a new project with a single command. Choose from multiple templates: default, minimal, professional, showcase, or glass.

🧩

27+ UI Components

Built-in components with pb- prefix and external glassmorphism components with pg- prefix. Add them individually or as a package.

🪟

Glassmorphism Ready

Frosted glass surfaces with backdrop-filter: blur(), semi-transparent backgrounds, and gradient accents — no CSS expertise needed.

📦

No Build Step

Pinkboba generates plain HTML, CSS, and JS. No bundlers, no frameworks, no configuration. Just open index.html and go.

🎨

Dark Theme by Default

All components ship with a modern dark theme. Toggle between light and dark mode with the built-in theme component.

🌐

External Packages

Install component packages from GitHub with pinkboba add glass. The package system downloads components at runtime — no recompilation needed.

Quick Start

Get a project running in under 30 seconds.

Install Pinkboba

Download the latest pinkboba.exe and add it to your PATH, or use cargo install pinkboba if you have Rust installed.

Create a project

pinkboba new my-app --template glass

This downloads the glass template and 27 glassmorphism components.

Add components

cd my-app
pinkboba add navbar
pinkboba add card
pinkboba add theme

Each component injects CSS, JS, and HTML into your project.

Preview

pinkboba serve

Open your browser and see your project live.
