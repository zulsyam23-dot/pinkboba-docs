Glass Package

27 glassmorphism UI components with frosted glass surfaces, blur effects, and gradient accents.

Getting Started CLI Reference Templates Components Glass Package

Overview

The glass package is an external component collection that brings glassmorphism design to your Pinkboba projects. All components use the pg- prefix and feature:

  • backdrop-filter: blur(20px) — frosted glass effect
  • Semi-transparent rgba() backgrounds
  • Gradient accent colors (purple #7c6cf0 and pink #f06c9a)
  • Dark theme optimized color schemes
  • Subtle border highlights with rgba(255,255,255,0.08)

The package is hosted on GitHub and downloaded at runtime — no recompilation needed.

Installation

Terminal
# Install all glass components in an existing project
pinkboba add glass

Or create a new project with the glass template (includes all 27 components):

pinkboba new my-app --template glass

Glass Styling Reference

Every glass component follows this styling pattern:

/* Core glass effect */
background: rgba(255, 255, 255, 0.06);
backdrop-filter: blur(20px);
-webkit-backdrop-filter: blur(20px);
border: 1px solid rgba(255, 255, 255, 0.08);
border-radius: 12px;

/* Text colors */
color: #e8e8f0;

/* Accent gradient */
background: linear-gradient(135deg, #7c6cf0, #f06c9a);

Offline Development

To use the glass package without internet access, clone the repository and set the environment variable:

Terminal
git clone https://github.com/zulsyam23-dot/pinkboba-components.git

# PowerShell
$env:PINKBOBA_PACKAGES_PATH = "C:\path\to\pinkboba-components"

# CMD
set PINKBOBA_PACKAGES_PATH=C:\path\to\pinkboba-components

Pinkboba will look for packages at $PINKBOBA_PACKAGES_PATH/glass/ instead of fetching from GitHub.
