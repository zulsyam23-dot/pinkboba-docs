v0.1.0 — 28 Mei 2026

Rilis perdana Pinkboba. Total 227 tes (108 unit × 2 binary + 11 integrasi), 0 failure.

Fitur Baru

  • DB Admin Studio — pinkboba admin untuk eksplorasi database SQLite via CLI maupun Web UI (port 9090). Fitur: CRUD rows, search, pagination, SQL editor, table manager (create/drop), dark/light theme, server monitoring.
  • Dokumentasi Website — pinkboba-docs dengan 7 halaman: Getting Started, CLI Reference (25 frontend + 11 backend commands), Frontend Components, Backend Generator, Templates, Glass Package, Download.
  • 227 Test Suite — 108 unit test (43 admin + 25 backhead generator + 26 UI components + 15 templates + 4 injection + 7 pinkboba file), 11 integration test (scaffold+build, pipeline, component generation).

Bug Fixes

  • Fix: pinkboba serve crash saat cargo tidak ada — Sebelumnya pinkboba serve langsung error jika cargo tidak ada di PATH atau build backend gagal. Sekarang tetap lanjut pakai binary cached di dist/backhed.exe atau fallback ke frontend-only di src/.
  • Fix: pinkboba serve fallback ke port 8080 — Jika backend binary sudah ada, serve sekarang langsung ke full-stack mode di port 3000 tanpa perlu rebuild ulang.
  • Fix: Build access denied — dist/ yang terkunci oleh proses sebelumnya tidak lagi menghentikan seluruh serve process.
  • Fix: Typo dokumentasi — Semua typo pinkboda → pinkboba dan Creat → Create di dokumentasi telah diperbaiki.
  • Fix: Port conflict — Jika port 3000 sudah dipakai, tidak crash tapi memberikan pesan yang jelas.
