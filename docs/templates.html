Default Template

A clean starter template with injection markers ready for components. No extra styling, no assumptions — just the essentials.

index.html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>{{PROJECT_NAME}}</title>
  <!--INJECT_THEME-->
  <!--INJECT_CSS-->
</head>
<body>
  <!--INJECT_HTML-->
  <!--INJECT_JS-->
</body>
</html>

Use with:

pinkboba new my-app

Minimal Template

The smallest possible starting point. Single file structure with minimal boilerplate. Great for experiments or when you want full control.

Use with:

pinkboba new my-app --template minimal

Professional Template

A landing-page-ready template with gradient hero section, feature cards, and responsive layout. Includes a purple-to-blue gradient accent scheme.

Use with:

pinkboba new my-app --template professional

Showcase Template

A full-featured multi-page demo template with client-side routing, animated counters, dark/light theme toggle, and 25+ pre-installed pb- components. Best for demonstrating the full capability of Pinkboba.

Features

  • 5 routes: Home, Features, Components, Pricing, About
  • Client-side SPA routing with data-route attributes
  • Intersection Observer animated counters
  • Theme toggle with localStorage persistence
  • All 25 built-in components pre-loaded

Use with:

pinkboba new my-app --template showcase

Glass Template

The glass template is an external template downloaded from the pinkboba-components GitHub repository at runtime. It includes:

  • Glassmorphism-themed HTML, CSS, and JS
  • All 27 pg-* glass components
  • Frosted glass surfaces with backdrop-filter: blur()
  • Gradient accent colors

Use with:

pinkboba new my-app --template glass
Note: The glass template requires an internet connection on first use. It downloads from https://raw.githubusercontent.com/zulsyam23-dot/pinkboba-components. Set PINKBOBA_PACKAGES_PATH for offline use.
