How Components Work

Each component is a .pinkboba file containing three sections separated by markers:

---STYLE---
/* CSS for this component */
.pb-button { ... }

---SCRIPT---
// JavaScript for this component
document.querySelectorAll('.pb-button')...

---TEMPLATE---
<!-- HTML template for this component -->
<button class="pb-button">...</button>

When you run pinkboba add <component>, the tool:

  1. Reads the .pinkboba file
  2. Extracts the CSS and appends it to src/style.css
  3. Extracts the JS and appends it to src/script.js
  4. Inserts the HTML into index.html at the <!--INJECT_HTML--> marker

Using Components

# Add individual components
pinkboba add navbar
pinkboba add hero
pinkboba add card
pinkboba add button
pinkboba add theme  # adds dark/light theme toggle

# Add the full glass package (27 components)
pinkboba add glass

Removing Components

pinkboba remove component navbar

This removes the component's CSS, JS, and HTML from your project files.

Previewing Components

Generate a component's output without adding it to the project:

pinkboba component button

External Components (pg-*)

The glass package provides 27 additional components with the pg- prefix, featuring glassmorphism styling. These are downloaded from GitHub at runtime.
