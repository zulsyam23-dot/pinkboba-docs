Getting Started

Install Pinkboba and build your first project in minutes.

Getting Started CLI Reference Templates Components Glass Package

1. Install Pinkboba

The easiest way to install Pinkboba is via Cargo (Rust package manager):

Terminal
cargo install pinkboba

If you don't have Rust installed, download the binary directly.

2. Create a project

Use the new command to scaffold a project. Choose a template with the --template flag:

Terminal
pinkboba new my-project

This creates a new project using the default template. To use the glassmorphism template:

pinkboba new my-glass-app --template glass

3. Navigate into the project

cd my-project

4. Add components

Components are the building blocks of your UI. Add them one at a time:

pinkboba add navbar
pinkboba add hero
pinkboba add button
pinkboba add card
pinkboba add theme

Each component injects its CSS, JavaScript, and HTML markup into your project files automatically.

5. Preview

Start a local development server:

Terminal
pinkboba serve

Open http://localhost:8080 in your browser. The server watches for file changes and refreshes automatically.

6. Build for production

Generate a production-ready build:

pinkboba build

This creates a dist/ folder with all assets bundled.
