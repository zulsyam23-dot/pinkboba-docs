Installation Options

Option 1: Via Cargo (requires Rust)

Terminal
cargo install pinkboba

Option 2: Manual (no Rust)

  1. Download pinkboba.exe from the button above
  2. Place it in a folder, e.g. C:\tools\
  3. Add that folder to your PATH environment variable
  4. Open a new terminal and run pinkboba --help

Option 3: Build from source

Terminal
git clone https://github.com/zulsyam23-dot/pinkboba.git
cd pinkboba
cargo install --path .
Note: Pinkboba downloads components from GitHub at runtime. An internet connection is required for pinkboba add glass and pinkboba new --template glass. Set the PINKBOBA_PACKAGES_PATH environment variable for offline development.
